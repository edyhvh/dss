Deadseascrolls is a display typeface designed for Hebrew text, with a style inspired by historical manuscript forms and optimized for modern digital usage. This release focuses on Hebrew support without niqqud, and includes a basic Latin set for interoperability in user interfaces.

Deadseascrolls is co-owned and co-authored by Jhonny and Adi Greenberg.

To contribute, see github.com/edyhvh/dss.
