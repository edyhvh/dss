# Dead Sea Scrolls Font Project

Dead Sea Scrolls is a single-style display Hebrew typeface inspired by historical manuscript forms.

## Ownership

- Co-owners/co-authors: Adi Greenberg and Jhonny Vergara

## Repository layout

- Root submission files:
  - `Deadseascrolls-Regular.ttf`
  - `METADATA.pb`
  - `DESCRIPTION.en_us.html`
  - `OFL.txt`
  - `SUBMISSION.md`
- `docs/` contains process notes, checklist, and issue draft.
- `legacy/` contains non-immediate-submission and archival files.
